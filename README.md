Teste usando o VS Code, com Claude Code e Ollama

## Esquema de Cores

Tema **deep space** — fundo ultraescuro com acentos em ciano, azul elétrico e roxo.

### Fundo / Base

| Cor | Uso |
| --- | --- |
| `#0B0B1E` | Background principal (HTML) — roxo-escuro profundo |
| `#000000` | Background do body (CSS) |

### Acentos / Destaque

| Cor | Uso |
| --- | --- |
| `#00F2FE` | Ciano brilhante — anel giratório, gradientes |
| `#00C3FF` | Azul claro — botões / CTA |
| `#9B51E0` | Roxo vibrante — gradientes do HTML |
| `#6A1B9A` | Roxo escuro — gradientes do HTML |

### Superfícies (Three.js)

| Cor | Uso |
| --- | --- |
| `0x2A8A2A` / emissivo `0x1A4A1A` | Planeta (verde + brilho verde-escuro) |
| `0xAAAAAA` | Lua (cinza claro) |

### Texto / Neutros

| Cor | Uso |
| --- | --- |
| `#FFFFFF` / `0xFFFFFF` / `white` | Textos, estrelas, luz principal |
| `#B8B8D4` | Lavanda claro — textos secundários do HTML |
| `0x333333` | Cinza escuro — luz ambiente, trilha da órbita |
| `0x222222` | Cinza ainda mais escuro — trilha da órbita da lua |
